Meus Projetos

E2B Windows Autoinstall

Um conjunto de arquivos de configuração e batch scripts para automatizar a instalação do Windows 10 e 11.
Este projeto tem como objetivo auxiliar o usuário no processo de instalação do Windows, pulando etapas de configuração e instalando programas básicos do dia-a-dia automaticamente, afim de facilitar e acelerar a tarefa, podendo ser utilizado em qualquer máquina, indiferente do hardware.

https://github.com/mcarneir0/e2b-windows-autoinstall

Clonezilla Live SME

Clonezilla personalizado para os servidores da SME.
Baseado no Clonezilla, programa para criação de imagens e clonagem de discos rígidos (HD/SSD).
Contém modificações feitas para automatizar a conexão com o servidor de imagens da Secretaria Municipal da Educação de Fortaleza.

https://github.com/mcarneir0/clonezilla-live-amd64-SME

Eobot Robot (Descontinuado)

Um aplicativo que monitora a cotação do site Eobot e decide qual a melhor criptomoeda para minerar no próprio site, enviando alertas através de um bot no Telegram.
Foi escrito em Java e seu funcionamento consistia no consumo de APIs REST do Eobot e Telegram.
Este aplicativo foi descontinuado devido ao encerramento das atividades do Eobot, confira mais informações no repositório.

https://github.com/mcarneir0/eobot-robot-repo
