Meus Projetos

Fortigate Config Backup Tool

Um script voltado para administradores de redes com o objetivo de auxiliar na realização de backups do arquivo de configuração de firewalls Fortigate. Totalmente feito em Python e utilizando a API REST local de cada appliance.

https://github.com/mcarneir0/fortigate-backup-api

Clonezilla Live SME

Clonezilla personalizado para os servidores da SME.
Baseado no Clonezilla, programa para criação de imagens e clonagem de discos rígidos (HD/SSD).
Contém modificações feitas para automatizar a conexão com o servidor de imagens da Secretaria Municipal da Educação de Fortaleza e atalhos para opções de clonagem.

https://github.com/mcarneir0/clonezilla-live-amd64-SME

E2B Windows Autoinstall

Um conjunto de arquivos de configuração e batch scripts para automatizar a instalação do Windows 10 e 11.
Este projeto tem como objetivo auxiliar o usuário no processo de instalação do Windows, pulando etapas de configuração e instalando programas básicos do dia-a-dia automaticamente, podendo ser utilizado em qualquer máquina, indiferente do hardware.

https://github.com/mcarneir0/e2b-windows-autoinstall

Eobot Robot (Descontinuado)

Um bot que monitora a cotação do Bitcoin no site Eobot e decide qual o melhor momento para minerar no próprio site, enviando alertas através de um bot no Telegram.
Foi escrito em Java e seu funcionamento consistia em consultas às APIs REST do Eobot e Telegram.
Este projeto foi descontinuado devido ao encerramento das atividades do Eobot, confira mais informações no repositório abaixo.

https://github.com/mcarneir0/eobot-robot-repo
